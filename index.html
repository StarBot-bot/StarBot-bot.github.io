Features
Here are all the features of StarBot
Fun

There are loads of fun commands like 8ball, coinflip, meme, dog, cat, airpods, lenny and more! Use gayrate and simprate on other users to make the bot rate them! When using the number command, the bot will generate a random number from 1 to 10 000 and send it!

Utility

There's a cool command named weather that shows you current weather information about places all around the world! Then commands like serverinfo(shows info about the server) and userinfo(shows info about an user). There are avatar, color, servericon and math command too!

Moderation

StarBot has simple moderation commands such as ban that bans an user from the server with a reason! The bot has a kick command too. There's an useful command named purge that deletes an amount of messages (up to 100)!
