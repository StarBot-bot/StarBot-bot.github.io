Features
Here are all the features of StarBot
Fun

There are loads of fun commands like 8ball, coinflip, lenny and more! Use gayrate and simprate on other users to make the bot rate them! When using the number command, the bot will generate a random number from 1 to 10 000 and send it!

Utility

There's a cool command named weather that shows you current weather information about places all around the world! Then commands like serverinfo(shows info about the server) and userinfo(shows info about an user). There are avatar, color, servericon and math command too!

Moderation

StarBot has simple moderation commands such as ban that bans an user from the server with a reason! The bot has a kick command too. StarBot comes with a simple warn system with the commands warn, warns and delallwarns.

Image

Image commands are commands that send some images, we have cat, dog and fox commands that send a random picture of the animal. We have a meme command that sends a random meme and there's the airpods command that sends some airpods picture with your text too!

Config

Config is used to config some things in the guild. For now you can configure the logging channel and some stuff about logging.
